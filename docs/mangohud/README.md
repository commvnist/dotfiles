# MangoHud

GNU Stow package for MangoHud overlay configuration.

## Dependencies

```sh
sudo pacman -S --needed mangohud
```

No AUR package is required.

## Install

From the repository root:

```sh
stow mangohud
```

This links `mangohud/.config/MangoHud/MangoHud.conf` to
`~/.config/MangoHud/MangoHud.conf`.
